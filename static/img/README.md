# Textbook Images and Assets

This directory contains images and assets for the Physical AI & Humanoid Robotics textbook.

## Image Categories

### Module 1: Robotic Nervous System (ROS 2)
- `ros2-architecture.png` - Diagram of ROS 2 architecture
- `ros2-nodes-topics-services.png` - Visualization of nodes, topics, and services
- `urdf-robot-model.png` - Example URDF robot model

### Module 2: Digital Twin (Gazebo & Unity)
- `gazebo-simulation.png` - Example Gazebo simulation environment
- `unity-robotics.png` - Unity robotics integration example
- `sensor-simulation.png` - Sensor simulation visualization

### Module 3: AI-Robot Brain (NVIDIA Isaac)
- `isaac-sim-photorealistic.png` - Isaac Sim photorealistic rendering
- `isaac-ros-pipeline.png` - Isaac ROS processing pipeline
- `nav2-architecture.png` - Navigation2 system architecture

### Module 4: Vision-Language-Action (VLA)
- `vla-system-architecture.png` - VLA system diagram
- `voice-action-flow.png` - Voice-to-action processing flow
- `cognitive-planning.png` - Cognitive planning visualization

### General Assets
- `logo.svg` - Textbook logo
- `favicon.ico` - Website favicon
- `docusaurus-social-card.jpg` - Social sharing image

## Image Specifications
- Format: PNG, JPEG, or SVG for diagrams
- Resolution: High enough for clear display (minimum 72 DPI)
- Size: Optimized for web delivery
- Accessibility: Include alt text descriptions
